# [228. Summary Ranges](https://leetcode.com/problems/summary-ranges/)

## 题目
Given a sorted integer array without duplicates, return the summary of its ranges.

Example 1:
```
Input: [0,1,2,4,5,7]
Output: ["0->2","4->5","7"]
```

Example 2:
```
Input: [0,2,3,4,6,8,9]
Output: ["0","2->4","6","8->9"]
```
